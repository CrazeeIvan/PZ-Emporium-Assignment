Installation Instructions

All mods are available on the Steam Workshop. Simply subscribe to the mods you want, and they'll automatically download and install. For manual installation:

  1. Download the mod file from the workshop or GitHub
  2. Navigate to your Project Zomboid mods folder
  3. Extract the mod folder into the mods directory
  4. Enable the mod in the game's mod menu
  5. Restart the game for changes to take effect
